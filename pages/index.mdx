# Overview

Artifacts is a sandbox MMORPG designed for code lovers, where you control your characters entirely through an HTTP API. Every action your characters can take corresponds to an endpoint you can interact with programmatically. This unique approach lets you automate gameplay, build custom tools, and explore new ways to play.

![Characters](https://artifactsmmo.com/images/docs/characters.png)

The API is fully documented using OpenAPI ([view the docs here](https://api.artifactsmmo.com/docs/#/)).
If you want to get started quickly, check out our [Quickstart Guide](/quickstart/introduction).

## Gameplay

Artifacts is a sandbox MMORPG with no predetermined pathâ€”you choose your own adventure. Send your characters into combat against monsters, gather resources using skills like mining and woodcutting, craft equipment, complete quests, and much more.

![Game](https://artifactsmmo.com/images/docs/scgame1.png)

The game's economy is player-driven: players can sell and buy items on the Grand Exchange.

Explore our [gameplay concepts](/concepts/actions) for a deeper dive, or jump right in with the [quickstart guide](/quickstart/introduction).

Artifacts is currently in alpha and under active development, with new features and content being added regularly.

## Features

- Elemental combat system (fire, air, water, earth)
- 50 levels of combat and skills
- 8 skills: Mining, Woodcutting, Cooking, Alchemy, Gearcrafting, Weaponcrafting, Jewelrycrafting
- Grand Exchange for trading items with other players
- Banking system
- Dynamic world events (exclusive monsters, resources, and NPCs that appear randomly for limited times)
- A dynamic tasks system for earning exclusive items
- Achievements system

Currently, the game features **over 450 items, 45 monsters, 300+ crafting recipes, and 400 maps**.

A game client with a built-in JavaScript editor is also available, making it easy to experiment and discover the game in the best possible conditions.
