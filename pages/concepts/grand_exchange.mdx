# Grand Exchange
 
The Grande Exchange lets you buy and sell items from other players. Item prices increase and decrease according to buy and sell orders. 

You can view all items for sale and their prices using this request. 

```http filename="cURL" copy
curl --location --request GET 'https://api.artifactsmmo.com/ge/?page=1&size=50' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
```
```js filename="Javascript" copy
var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   redirect: 'follow'
};

fetch("https://api.artifactsmmo.com/ge/?page=1&size=50", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
```

[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/all_ge_items_ge__get)


## Buy/sell item

To buy or sell an item, you must be on a map containing a Grand Exchange.

You can use this POST request to purchase an item:

```http filename="cURL" copy
curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/ge/buy \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer INSERT_YOUR_TOKEN_HERE' \
  --header 'Content-Type: application/json' \
  --data '{
  "code": "string",
  "quantity": 1,
  "price":1
}'
```
```js filename="Javascript" copy
const url = 'https://api.artifactsmmo.com/my/{name}/action/ge/buy';
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer INSERT_YOUR_TOKEN_HERE'
  },
  body: '{"code":"string","quantity":1,"price":1}'
};

try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
```
[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/action_ge_buy_item_my__name__action_ge_buy_post)

```http filename="cURL" copy

  --url https://api.artifactsmmo.com/my/{name}/action/ge/sell \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer INSERT_YOUR_TOKEN_HERE' \

  "code": "string",
```
