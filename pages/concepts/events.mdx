# Events

Events let exclusive monsters and resources appear randomly in the world. When an event appears, you have a limited amount of time to get to the map to kill the monster or collect the resource.

| Name               | Content                              |
| ----------------   | ---------------------------------- |
| Bandit Camp        |[Bandit Lizard (Monster)](https://artifactsmmo.com/encyclopedia/monsters/bandit_lizard) |
| Portal (Demon)            |[Demon (Monster)](https://artifactsmmo.com/encyclopedia/monsters/demon)|
| Portal (Efreet Sultan)             |[Demon (Efreet Sultan)](https://artifactsmmo.com/encyclopedia/monsters/efreet_sultan)|
| Cult of Darkness   |[Cultist Emperor (Monster)](https://artifactsmmo.com/encyclopedia/monsters/cultist_emperor)|
| Strange Apparition |[Strange Rocks (Resource)](https://artifactsmmo.com/encyclopedia/resources/strange_rocks)|
| Magic Apparition   |[Magic Tree (Resource)](https://artifactsmmo.com/encyclopedia/resources/magic_tree)|
| Rosenblood         |[Rosenblood (Monster)](https://artifactsmmo.com/encyclopedia/monsters/rosenblood)|
| Snowman         |[Snowman (Monster)](https://artifactsmmo.com/encyclopedia/monsters/snowman)|

You can view all events using this request :

```http filename="cURL" copy
curl --location --request GET 'https://api.artifactsmmo.com/events/' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
```
```js filename="Javascript" copy
var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   redirect: 'follow'
}; 

fetch("https://api.artifactsmmo.com/events/", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
```

[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/get_all_events_events_get)
