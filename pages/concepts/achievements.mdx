# Achievements

Achievements are objectives to be met during the season by all your characters. They grant you gold and achievement points, which allow you to earn season badges.

![Client](https://artifactsmmo.com/images/docs/badges_msg.png)

> **Note:** Achievements are tracked at the account level. All your characters contribute together to unlock achievements.

Achievement points and the number of achievements required to complete a season may change. You can always check the current requirements and available badges using the [server status endpoint](/api_guide/openapi_spec) or the `/badges` endpoint.

## Achievement Types

- `combat_kill` – Defeat monsters
- `combat_drop` – Obtain item drops from combat
- `combat_level` – Reach certain combat levels
- `gathering` – Gather resources
- `crafting` – Craft items
- `recycling` – Recycle equipment
- `task` – Complete tasks
- `other` – Miscellaneous achievements
- `use` – Use consumables

## Querying Achievements

You can use this **GET** request to view your account’s achievements and their progress:

```http filename="cURL" copy
curl --location --request GET 'https://api.artifactsmmo.com/accounts/{account}/achievements?type=&completed=&page=1&size=50' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
```

Query parameters:
- `type`: Filter by achievement type (see list above)
- `completed`: true/false – completed or not
- `page`/`size`: Pagination

Example response:
```json
{
  "data": [
    {
      "name": "Monster Slayer",
      "code": "slay_100_monsters",
      "description": "Defeat 100 monsters.",
      "points": 5,
      "type": "combat_kill",
      "target": "monster",
      "total": 100,
      "current": 42,
      "completed_at": null,
      "rewards": { "gold": 100 }
    }
    // ...
  ]
}
```

[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/get_account_achievements_accounts__account__achievements_get)

You can also view the complete list of possible achievements with:

```http
GET /achievements
```

## Season Badges

Each season, accumulating enough achievement points can earn you unique seasonal badges.  
You can view the current badges and requirements via the `/badges` endpoint or in the server details.
