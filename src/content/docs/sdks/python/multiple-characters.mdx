---
title: Multiple Characters
description: Run all 5 characters simultaneously using asyncio and the Python SDK.
---

Artifacts lets you control **up to 5 characters** per account. Running them in parallel requires the async client and Python's `asyncio`.

## Why async?

The sync client blocks your Python thread while waiting for cooldowns. If you have 5 characters each with 5-second cooldowns, a sequential sync program would take 25 seconds to complete one round — instead of 5.

With `asyncio`, all 5 characters wait for their cooldowns concurrently: the total time is just the longest individual cooldown.

## Basic pattern — `asyncio.gather()`

Define a coroutine for what one character should do, then launch all five at the same time:

```python
import asyncio
from artifacts import AsyncArtifactsClient

async def combat_loop(char, rounds: int = 10):
    """Make a character fight N rounds, resting when HP is low."""
    for _ in range(rounds):
        info = await char.get()
        if info.hp < info.max_hp * 0.3:
            await char.rest()
            continue
        result = await char.fight()
        print(f"[{char.name}] {result.fight.result.value}")

async def main():
    async with AsyncArtifactsClient(token="YOUR_TOKEN") as client:
        names = ["Char1", "Char2", "Char3", "Char4", "Char5"]
        chars = [client.character(n) for n in names]

        # Run all 5 coroutines concurrently
        await asyncio.gather(*[combat_loop(c, rounds=20) for c in chars])

asyncio.run(main())
```

## Different tasks per character

Characters don't have to do the same thing. Assign different coroutines to each one:

```python
async def gather_loop(char):
    """Move to a resource tile and gather indefinitely."""
    await char.move(x=-2, y=1)   # woodcutting tile (example)
    while True:
        await char.skills.gather()

async def bank_loop(char):
    """Deposit everything into the bank, then withdraw gold."""
    await char.move(x=4, y=1)   # bank tile (example)
    await char.bank.deposit_items([...])
    await char.bank.withdraw_gold(quantity=500)

async def main():
    async with AsyncArtifactsClient(token="YOUR_TOKEN") as client:
        fighter  = client.character("Fighter")
        gatherer = client.character("Gatherer")
        banker   = client.character("Banker")

        await asyncio.gather(
            combat_loop(fighter),
            gather_loop(gatherer),
            bank_loop(banker),
        )

asyncio.run(main())
```

## Infinite loops and graceful shutdown

For always-running bots, wrap the loop body in a `try/except` so a single error doesn't kill all characters:

```python
import asyncio
from artifacts import AsyncArtifactsClient
from artifacts.errors import ArtifactsAPIError

async def bot_loop(char):
    while True:
        try:
            info = await char.get()
            if info.hp < info.max_hp * 0.3:
                await char.rest()
            else:
                await char.fight()
        except ArtifactsAPIError as e:
            print(f"[{char.name}] API error [{e.code}]: {e.message} — retrying in 10s")
            await asyncio.sleep(10)
        except Exception as e:
            print(f"[{char.name}] Unexpected error: {e} — retrying in 30s")
            await asyncio.sleep(30)

async def main():
    async with AsyncArtifactsClient(token="YOUR_TOKEN") as client:
        names = ["Char1", "Char2", "Char3", "Char4", "Char5"]
        await asyncio.gather(*[bot_loop(client.character(n)) for n in names])

asyncio.run(main())
```

## Mixing sync and async in one project

If you only need to do quick one-off calls alongside your async bot, you can use two separate programs or run the sync client in a thread:

```python
import asyncio
from concurrent.futures import ThreadPoolExecutor
from artifacts import ArtifactsClient

def sync_check():
    with ArtifactsClient(token="YOUR_TOKEN") as client:
        return client.my_account.get_details()

async def main():
    loop = asyncio.get_event_loop()
    with ThreadPoolExecutor() as pool:
        account = await loop.run_in_executor(pool, sync_check)
        print(account.username)

asyncio.run(main())
```

:::tip[Rule of thumb]
For bots with multiple characters, always use `AsyncArtifactsClient`. For quick scripts or single-character automation, `ArtifactsClient` (sync) is simpler and equally effective.
:::
