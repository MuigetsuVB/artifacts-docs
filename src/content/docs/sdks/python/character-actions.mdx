All action methods are called on a **character controller** returned by `client.character("name")`. The SDK automatically waits for the cooldown after each action (unless you disable it — see [Cooldowns & Retry](/sdks/python/cooldowns-and-retry)).

```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("MyCharacter")
    # All examples below use `char`
```

## Movement

Characters navigate a 2D grid. Tile coordinates are integers (x, y). Each move triggers a cooldown.

```python
# Move by coordinates
char.move(x=2, y=3)

# Move by map ID (useful when you have a known tile from the maps API)
char.move(map_id=42)

# Layer transition — enter a building, dungeon, or other layer
char.transition()
```

:::tip[Finding coordinates]
Use `client.maps.get_all(content_type="monster", content_code="chicken")` to find the tile that contains the monster you want to fight. See [Game Data](/sdks/python/game-data).
:::

## Combat

### Solo fight

```python
result = char.fight()

fight = result.fight
print(f"Result: {fight.result.value}")          # 'win' or 'lose'
print(f"Turns: {fight.turns}")

for cr in fight.characters:
    print(f"  +{cr.xp} XP   +{cr.gold} gold   drops: {[d.code for d in cr.drops]}")
```

### Group (boss) fight

Up to 3 characters can fight together. Pass the names of the other participants:

```python
result = char.fight(participants=["Char2", "Char3"])
```

All participating characters must be on the same tile before the fight starts.

### Rest (recover HP)

```python
result = char.rest()
print(f"Restored {result.hp_restored} HP")
```

## Skills — Gathering, Crafting, Recycling

For a full guide, see the dedicated [Skills](/sdks/python/skills/) page.

The character must be standing on the correct tile (resource tile for gathering, workshop for crafting).

```python
# Gather the resource on the current tile
result = char.skills.gather()
print(f"+{result.details.xp} XP  got: {[d.code for d in result.details.items]}")

# Craft an item (character must be at the appropriate workshop)
char.skills.craft(code="iron_sword", quantity=1)

# Recycle equipment into raw materials
char.skills.recycle(code="iron_sword", quantity=1)
```

:::caution[XP cap]
If your skill level exceeds the resource or recipe level by more than 10, you earn **0 XP**. Plan your grinding accordingly.
:::

## Equipment

For a full guide, see the dedicated [Equipment](/sdks/python/equipment/) page.

```python
from artifacts.models.enums import ItemSlot

# Equip an item from inventory into a slot
char.equipment.equip(code="iron_sword", slot=ItemSlot.WEAPON)

# Unequip the item currently in a slot (it returns to inventory)
char.equipment.unequip(slot=ItemSlot.WEAPON)
```

Available `ItemSlot` values: `WEAPON`, `RUNE`, `SHIELD`, `HELMET`, `BODY_ARMOR`, `LEG_ARMOR`, `BOOTS`, `RING1`, `RING2`, `AMULET`, `ARTIFACT1`, `ARTIFACT2`, `ARTIFACT3`, `UTILITY1`, `UTILITY2`.

## Bank

For a full guide, see the dedicated [Bank](/sdks/python/bank/) page.

The character must be standing on a bank tile.

```python
# Gold deposits and withdrawals
char.bank.deposit_gold(quantity=500)
char.bank.withdraw_gold(quantity=200)

# Item deposits and withdrawals
from artifacts.models.common import SimpleItemSchema

items = [SimpleItemSchema(code="copper_ore", quantity=50)]
char.bank.deposit_items(items)
char.bank.withdraw_items(items)

# Buy a bank expansion (+20 item slots, costs gold)
char.bank.buy_expansion()
```

## Grand Exchange (GE)

For a full guide, see the dedicated [Grand Exchange](/sdks/python/grand-exchange/) page.

The character must be standing on a GE tile.

```python
# Post a sell order (your items are locked until sold or cancelled)
char.ge.sell(code="copper_ore", quantity=100, price=5)

# Buy from an existing sell order
char.ge.buy(id="order_id_here", quantity=10)

# Post a buy order (your gold is locked upfront)
char.ge.create_buy_order(code="copper_ore", quantity=100, price=5)

# Fill someone else's buy order (deliver items, receive gold)
char.ge.fill(id="order_id_here", quantity=50)

# Cancel your own order (items or gold returned to you)
char.ge.cancel(id="order_id_here")
```

## Tasks

For a full guide, see the dedicated [Tasks](/sdks/python/tasks/) page.

The character must be at a task master.

```python
# Accept a new task from the task master
result = char.tasks.new()
print(f"Task: {result.task.code}  type={result.task.type.value}  goal={result.task.total}")

# Trade task items in (partial progress is allowed)
char.tasks.trade(code="copper_ore", quantity=10)

# Complete the task when fully done
char.tasks.complete()

# Exchange 6 task coins for a random reward
char.tasks.exchange()

# Cancel the current task (costs 1 task coin)
char.tasks.cancel()
```

## Giving items or gold to another character

For a full guide, see the dedicated [Trading](/sdks/python/trading/) page.

Both characters must be on the same tile.

```python
# Transfer gold
char.trading.give_gold(quantity=100, character="OtherChar")

# Transfer items
from artifacts.models.common import SimpleItemSchema

items = [SimpleItemSchema(code="copper_ore", quantity=20)]
char.trading.give_items(items=items, character="OtherChar")
```

## Miscellaneous

```python
# Claim a pending item (rewards from achievements, GE purchases, etc.)
char.claim_item(id=123)

# Change character skin
from artifacts.models.enums import CharacterSkin
char.change_skin(skin=CharacterSkin.MEN2)

# View character action logs
logs = char.get_logs(page=1, size=20)
for log in logs.data:
    print(f"[{log.type.value}] {log.description}")
```
