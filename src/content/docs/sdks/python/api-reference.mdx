This page lists every method available in the `artifacts-mmo` SDK. Both `ArtifactsClient` and `AsyncArtifactsClient` expose the same interface — async methods return coroutines instead of values.

## Top-level client accessors

| Accessor | Description |
|---|---|
| `client.server` | Server status |
| `client.token` | Token generation |
| `client.accounts` | Account management |
| `client.my_account` | Your own account details |
| `client.characters` | Character management |
| `client.items` | Item data |
| `client.monsters` | Monster data |
| `client.maps` | Map tile data |
| `client.resources` | Gatherable resource data |
| `client.npcs` | NPC data and shops |
| `client.events` | Game events |
| `client.achievements` | Achievement data |
| `client.badges` | Badge data |
| `client.effects` | Effects data |
| `client.grand_exchange` | GE market data (read-only) |
| `client.leaderboard` | Leaderboard data |
| `client.tasks` | Task definitions and rewards |
| `client.simulation` | Fight simulator (Members only) |
| `client.sandbox` | Sandbox helpers (Sandbox server only) |

## client.server

| Method | Description |
|---|---|
| `get_status()` | Returns the current server status and version |

## client.token

| Method | Parameters | Description |
|---|---|---|
| `generate(username, password)` | `str, str` | Generate a new JWT token for the account |

## client.accounts

| Method | Parameters | Description |
|---|---|---|
| `create(username, password, email)` | `str, str, str` | Create a new account |
| `get(username)` | `str` | Get public info for any account |
| `get_achievements(username)` | `str` | List achievements for an account |
| `get_characters(username)` | `str` | List characters for an account |

## client.my_account

| Method | Parameters | Description |
|---|---|---|
| `get_details()` | — | Your account details (gems, username, etc.) |
| `get_bank()` | — | Your bank info (gold balance) |
| `get_bank_items(page, size)` | optional | Items stored in your bank |
| `get_ge_orders(page, size)` | optional | Your active GE orders |
| `get_ge_history(page, size)` | optional | Your GE order history |
| `get_pending_items()` | — | Items waiting to be claimed |
| `get_characters()` | — | All characters on your account |
| `get_all_logs(page, size)` | optional | All action logs across all characters |
| `change_password(password, new_password)` | `str, str` | Change account password |

## Character — direct methods

Obtained via `char = client.character("name")`.

| Method | Parameters | Description |
|---|---|---|
| `get()` | — | Fetch current character state |
| `get_logs(page, size)` | optional | Character action log |
| `move(x, y)` | `int, int` | Move to coordinates |
| `move(map_id)` | `int` | Move to a tile by ID |
| `transition()` | — | Layer transition (enter/exit building) |
| `fight(participants, wait)` | optional | Start a fight; optionally group with other characters |
| `rest()` | — | Rest to recover HP |
| `claim_item(id)` | `int` | Claim a pending item |
| `change_skin(skin)` | `CharacterSkin` | Change character skin |

## Simulation (Members only)

```python
from artifacts.models.sandbox import FakeCharacterSchema

fake = FakeCharacterSchema(level=20, weapon_slot="iron_sword", body_armor_slot="iron_armor")
result = client.simulation.fight(characters=[fake], monster="ogre", iterations=100)
print(f"Winrate: {result.winrate:.1%}  ({result.wins}W / {result.losses}L)")
```

## Sandbox (Sandbox server only)

Only available when `base_url="https://api.sandbox.artifactsmmo.com"`.

| Method | Parameters | Description |
|---|---|---|
| `client.sandbox.give_gold(character, quantity)` | `str, int` | Give gold to a character |
| `client.sandbox.give_item(character, code, quantity)` | `str, str, int` | Give an item to a character |
| `client.sandbox.give_xp(character, skill, xp)` | `str, str, int` | Give XP to a character skill |
| `client.sandbox.spawn_event(event_code)` | `str` | Spawn a game event |
| `client.sandbox.reset_account()` | — | Reset the entire sandbox account |
