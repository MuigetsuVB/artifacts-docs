The SDK provides two account-related accessors:

- **`client.accounts`** — public endpoints (no auth required for reading, auth for creating)
- **`client.my_account`** — authenticated endpoints for your own account

```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    # All examples below use `client`
```

---

## Create an Account

```python
result = client.accounts.create(
    username="MyNewAccount",
    password="secure_password",
    email="email@example.com",
)
print(result)  # "OK"
```

## Get Public Account Info

Retrieve public details for any account — no token required:

```python
account = client.accounts.get("SomePlayer")
print(f"{account.username}  status={account.status.value}")
```

Returns an `AccountDetails` model with public fields like username, status, and creation date.

## Get Account Achievements

```python
from artifacts.models.enums import AchievementType

# All achievements for an account
achievements = client.accounts.get_achievements("SomePlayer")
for a in achievements.data:
    print(f"  {a.code}  completed={a.completed}")

# Filter by type
combat_achievements = client.accounts.get_achievements(
    "SomePlayer",
    type=AchievementType.COMBAT,
    completed=True,
)
```

Returns `DataPage[AccountAchievementSchema]` — see [Pagination](#pagination).

## Get Account Characters

```python
characters = client.accounts.get_characters("SomePlayer")
for char in characters:
    print(f"  {char.name}  lv{char.level}")
```

Returns a `list[CharacterSchema]` (not paginated).

## Password Recovery

```python
# Request a password reset email
client.accounts.forgot_password("email@example.com")

# Reset with the token received by email
client.accounts.reset_password(token="reset_token_here", new_password="new_secure_password")
```

## My Account (Authenticated)

All `client.my_account` endpoints require a valid token.

### Account Details

```python
details = client.my_account.get_details()
print(f"{details.username}  gems={details.gems}")
```

Returns `MyAccountDetails` with full account info including gems, subscription status, etc.

### Bank Info

Read your bank state without moving a character to a bank tile:

```python
# Bank gold balance
bank = client.my_account.get_bank()
print(f"Gold in bank: {bank.gold}")

# Items stored in bank (paginated)
items = client.my_account.get_bank_items(page=1, size=100)
for item in items.data:
    print(f"  {item.code} x{item.quantity}")

# Filter by specific item
iron = client.my_account.get_bank_items(item_code="iron_ore")
```

:::tip
`client.my_account.get_bank()` and `get_bank_items()` are **read-only** and don't require your character to be at a bank tile. To actually deposit or withdraw, use [char.bank](/sdks/python/bank/).
:::

### Grand Exchange Orders

```python
from artifacts.models.enums import GEOrderType

# All your active orders
orders = client.my_account.get_ge_orders()
for order in orders.data:
    print(f"  {order.code}  qty={order.quantity}  price={order.price}  type={order.type.value}")

# Filter by item or type
sell_orders = client.my_account.get_ge_orders(
    code="iron_ore",
    type=GEOrderType.SELL,
)

# GE transaction history
history = client.my_account.get_ge_history()
for tx in history.data:
    print(f"  {tx.code}  qty={tx.quantity}  price={tx.price}")

# Filter history by order ID or item code
history = client.my_account.get_ge_history(id="order_abc", code="iron_ore")
```

### Pending Items

Items waiting to be claimed (from achievements, GE purchases, etc.):

```python
pending = client.my_account.get_pending_items()
for item in pending.data:
    print(f"  id={item.id}  {item.code} x{item.quantity}  source={item.source.value}")
```

:::note
To claim a pending item, use `char.claim_item(id=...)` — see [Character Actions](/sdks/python/character-actions/).
:::

### My Characters

```python
chars = client.my_account.get_characters()
for c in chars:
    print(f"  {c.name}  lv{c.level}  pos=({c.x},{c.y})")
```

### All Action Logs

View a combined log of all actions across all your characters:

```python
logs = client.my_account.get_all_logs(page=1, size=50)
for log in logs.data:
    print(f"  [{log.type.value}] {log.description}")
```

### Change Password

```python
client.my_account.change_password(
    current_password="old_password",
    new_password="new_password",
)
```

:::caution
Changing your password **invalidates your current token**. You'll need to generate a new one.
:::

## Pagination

All paginated endpoints return a `DataPage[T]`:

```python
page = client.my_account.get_bank_items(page=1, size=50)

page.data    # list[SimpleItemSchema]
page.total   # int — total matching items
page.page    # int — current page (1-based)
page.pages   # int — total pages
page.size    # int — page size
```

## Full Method Reference

### client.accounts

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `create(username, password, email)` | `str, str, str` | `str` | Create a new account |
| `get(account)` | `str` | `AccountDetails` | Get public account info |
| `get_achievements(account, *, type, completed, page, size)` | `str`, optional | `DataPage[AccountAchievementSchema]` | Account achievements |
| `get_characters(account)` | `str` | `list[CharacterSchema]` | Account characters |
| `forgot_password(email)` | `str` | `str` | Request password reset |
| `reset_password(token, new_password)` | `str, str` | `str` | Reset password with token |

### client.my_account

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `get_details()` | — | `MyAccountDetails` | Your account details |
| `get_bank()` | — | `BankSchema` | Bank gold balance |
| `get_bank_items(*, item_code, page, size)` | optional | `DataPage[SimpleItemSchema]` | Items in bank |
| `get_ge_orders(*, code, type, page, size)` | optional | `DataPage[GEOrderSchema]` | Your GE orders |
| `get_ge_history(*, id, code, page, size)` | optional | `DataPage[GeOrderHistorySchema]` | Your GE history |
| `get_pending_items(*, page, size)` | optional | `DataPage[PendingItemSchema]` | Pending items |
| `get_characters()` | — | `list[CharacterSchema]` | Your characters |
| `get_all_logs(*, page, size)` | optional | `DataPage[LogSchema]` | All action logs |
| `change_password(current_password, new_password)` | `str, str` | `str` | Change password |
