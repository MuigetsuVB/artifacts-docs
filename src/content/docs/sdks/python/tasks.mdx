The task system lets characters accept quests from Task Masters, turn in required items, and earn rewards. The SDK provides:

- **`char.tasks`** — character actions (accept, trade, complete, exchange, cancel) — requires the character to be at a Task Master
- **`client.tasks`** — read-only data (browse available tasks and rewards)

```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("MyCharacter")
```

---

## Finding a Task Master

```python
task_masters = client.maps.get_all(content_type="tasks_master")
tile = task_masters.data[0]
char.move(x=tile.x, y=tile.y)
```

## Accept a New Task

```python
result = char.tasks.new()
task = result.task
print(f"Task: {task.code}  type={task.type.value}  total={task.total}")
```

Returns `TaskDataSchema` with the task details and cooldown.

## Trade Items (Partial Progress)

Turn in items towards your task goal. You can trade partial amounts:

```python
result = char.tasks.trade(code="copper_ore", quantity=10)
print(f"Progress: {result.task.progress}/{result.task.total}")
```

Returns `TaskTradeDataSchema`.

## Complete a Task

Once you've traded all required items, complete the task to receive task coins:

```python
result = char.tasks.complete()
print(f"Reward: {result.reward.code} x{result.reward.quantity}")
```

Returns `RewardDataSchema`.

## Exchange Task Coins

Exchange **6 task coins** for a random reward:

```python
result = char.tasks.exchange()
print(f"Got: {result.reward.code} x{result.reward.quantity}")
```

Returns `RewardDataSchema`.

## Cancel a Task

Cancel the current task. Costs **1 task coin**:

```python
result = char.tasks.cancel()
```

Returns `TaskCancelledSchema`.

## Browsing Task Data (Read-Only)

These endpoints don't require a character — they return game data about available tasks.

### List All Tasks

```python
from artifacts.models.enums import Skill, TaskType

# All tasks
all_tasks = client.tasks.get_all()
for t in all_tasks.data:
    print(f"  {t.code}  type={t.type.value}  lv{t.level}")

# Filter by type and level
monster_tasks = client.tasks.get_all(
    type=TaskType.MONSTERS,
    min_level=1,
    max_level=20,
)

# Filter by skill
mining_tasks = client.tasks.get_all(skill=Skill.MINING)
```

Returns `DataPage[TaskFullSchema]`.

### Get a Specific Task

```python
task = client.tasks.get("copper_ore")
print(f"{task.code}  type={task.type.value}  lv{task.level}")
```

### List Task Rewards

```python
rewards = client.tasks.get_all_rewards()
for r in rewards.data:
    print(f"  {r.code}  rate={r.rate}  min={r.min_quantity}  max={r.max_quantity}")
```

Returns `DataPage[DropRateSchema]`.

### Get a Specific Reward

```python
reward = client.tasks.get_reward("tasks_coin")
```

## Practical Example — Full Task Loop

```python
from artifacts.errors import TaskError, InventoryFullError

# Find task master and resource tiles
task_master = client.maps.get_all(content_type="tasks_master").data[0]

while True:
    # Go to task master and accept a new task
    char.move(x=task_master.x, y=task_master.y)
    result = char.tasks.new()
    task = result.task
    print(f"New task: collect {task.total}x {task.code}")

    # Find where to get the required items (example for gathering tasks)
    resource_tiles = client.maps.get_all(
        content_type="resource",
        content_code=task.code,
    )

    if resource_tiles.data:
        tile = resource_tiles.data[0]
        char.move(x=tile.x, y=tile.y)

        # Gather until we have enough
        gathered = 0
        while gathered < task.total:
            try:
                char.skills.gather()
                gathered += 1
            except InventoryFullError:
                break

    # Return to task master and trade items
    char.move(x=task_master.x, y=task_master.y)
    try:
        char.tasks.trade(code=task.code, quantity=task.total)
        char.tasks.complete()
        print("Task completed!")
    except TaskError as e:
        print(f"Task error [{e.code}]: {e.message}")
```

## Error Handling

All task-related API errors are caught by `TaskError` (codes 474–489):

```python
from artifacts.errors import TaskError, ArtifactsAPIError

try:
    char.tasks.complete()
except TaskError as e:
    print(f"Task error [{e.code}]: {e.message}")
except ArtifactsAPIError as e:
    print(f"API error [{e.code}]: {e.message}")
```

## Full Method Reference

### char.tasks (Character Actions)

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `new()` | — | `TaskDataSchema` | Accept a new task |
| `trade(*, code, quantity)` | `str, int` | `TaskTradeDataSchema` | Turn in items towards task |
| `complete()` | — | `RewardDataSchema` | Complete the task |
| `exchange()` | — | `RewardDataSchema` | Exchange 6 task coins for a reward |
| `cancel()` | — | `TaskCancelledSchema` | Cancel current task (costs 1 coin) |

### client.tasks (Read-Only Data)

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `get_all(*, min_level, max_level, skill, type, page, size)` | optional | `DataPage[TaskFullSchema]` | List task definitions |
| `get(code)` | `str` | `TaskFullSchema` | Get a single task |
| `get_all_rewards(*, page, size)` | optional | `DataPage[DropRateSchema]` | List all task rewards |
| `get_reward(code)` | `str` | `DropRateSchema` | Get a specific reward |
