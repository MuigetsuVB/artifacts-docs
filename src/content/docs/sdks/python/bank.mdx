The bank lets you store gold and items safely. The SDK offers two ways to interact with it:

- **`char.bank`** — character actions (deposit, withdraw, expand) — requires the character to be on a bank tile
- **`client.my_account`** — read-only access (check balance, list items) — works from anywhere

```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("MyCharacter")
```

---

## Finding a Bank Tile

Before depositing or withdrawing, your character must be standing on a bank tile:

```python
# Find all bank tiles
banks = client.maps.get_all(content_type="bank")
tile = banks.data[0]
print(f"Bank at ({tile.x}, {tile.y})")

# Move to the bank
char.move(x=tile.x, y=tile.y)
```

## Deposit Gold

```python
result = char.bank.deposit_gold(quantity=500)
print(f"Bank gold after deposit: {result.bank.gold}")
```

Returns `BankGoldTransactionSchema` containing the updated bank state and cooldown.

## Withdraw Gold

```python
result = char.bank.withdraw_gold(quantity=200)
print(f"Bank gold after withdrawal: {result.bank.gold}")
```

## Deposit Items

```python
from artifacts.models.common import SimpleItemSchema

items = [
    SimpleItemSchema(code="copper_ore", quantity=50),
    SimpleItemSchema(code="iron_ore", quantity=25),
]
result = char.bank.deposit_items(items)
```

Returns `BankItemTransactionSchema` with the updated bank inventory.

## Withdraw Items

```python
from artifacts.models.common import SimpleItemSchema

items = [SimpleItemSchema(code="copper_ore", quantity=20)]
result = char.bank.withdraw_items(items)
```

## Buy Bank Expansion

Each expansion adds **20 extra item slots** to your bank. Costs gold (price increases with each purchase).

```python
result = char.bank.buy_expansion()
print(f"New bank slots: {result.bank.slots}")
```

Returns `BankExtensionTransactionSchema`.

## Reading Bank State (No Movement Required)

You don't need to move your character to a bank tile to *read* bank data. Use the account-level endpoints:

```python
# Gold balance
bank = client.my_account.get_bank()
print(f"Gold: {bank.gold}  Slots: {bank.slots}")

# All items in bank (paginated)
page = client.my_account.get_bank_items(page=1, size=100)
for item in page.data:
    print(f"  {item.code} x{item.quantity}")

# Search for a specific item
iron = client.my_account.get_bank_items(item_code="iron_ore")
if iron.data:
    print(f"Iron ore in bank: {iron.data[0].quantity}")
else:
    print("No iron ore in bank")
```

## Practical Example — Auto-Bank Loop

A common pattern: gather resources, deposit when inventory is full, repeat.

```python
from artifacts.errors import InventoryFullError

# Find resource and bank tiles
resource_tile = client.maps.get_all(content_type="resource", content_code="copper_rocks").data[0]
bank_tile = client.maps.get_all(content_type="bank").data[0]

while True:
    # Move to resource and gather
    char.move(x=resource_tile.x, y=resource_tile.y)

    try:
        for _ in range(20):
            char.skills.gather()
    except InventoryFullError:
        pass

    # Move to bank and deposit everything
    char.move(x=bank_tile.x, y=bank_tile.y)

    info = char.get()
    items_to_deposit = [
        SimpleItemSchema(code=slot.code, quantity=slot.quantity)
        for slot in info.inventory
        if slot.code  # skip empty slots
    ]
    if items_to_deposit:
        char.bank.deposit_items(items_to_deposit)
        print(f"Deposited {len(items_to_deposit)} item stacks")
```

## Error Handling

```python
from artifacts.errors import (
    InsufficientGoldError,
    InventoryFullError,
    ContentNotOnMapError,
    ArtifactsAPIError,
)

try:
    char.bank.withdraw_gold(quantity=999999)
except InsufficientGoldError:
    print("Not enough gold in bank")

try:
    char.bank.withdraw_items([SimpleItemSchema(code="iron_ore", quantity=10)])
except ArtifactsAPIError as e:
    print(f"Bank error [{e.code}]: {e.message}")
```

## Full Method Reference

### char.bank (Character Actions)

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `deposit_gold(*, quantity)` | `int` | `BankGoldTransactionSchema` | Deposit gold into bank |
| `withdraw_gold(*, quantity)` | `int` | `BankGoldTransactionSchema` | Withdraw gold from bank |
| `deposit_items(items)` | `list[SimpleItemSchema]` | `BankItemTransactionSchema` | Deposit items into bank |
| `withdraw_items(items)` | `list[SimpleItemSchema]` | `BankItemTransactionSchema` | Withdraw items from bank |
| `buy_expansion()` | — | `BankExtensionTransactionSchema` | Buy +20 bank slots |

### client.my_account (Read-Only)

| Method | Parameters | Returns | Description |
|---|---|---|---|
| `get_bank()` | — | `BankSchema` | Bank gold balance and slot count |
| `get_bank_items(*, item_code, page, size)` | optional | `DataPage[SimpleItemSchema]` | Items stored in bank |
