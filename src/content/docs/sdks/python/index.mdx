The `artifacts-mmo` Python SDK covers all **65 API endpoints**, supports both sync and async usage, and is designed for people building bots and automation scripts.

- **Auto-cooldown** — waits automatically after each action; no manual `time.sleep()` needed
- **Auto-retry** — exponential backoff on rate limits, cooldown conflicts, and server errors
- **Type safety** — every response is a Pydantic model with full IDE autocompletion
- **Sync + async** — beginners use `ArtifactsClient`, advanced users use `AsyncArtifactsClient`
- **65 / 65 endpoints** — compatible with Artifacts MMO API v7

| | Raw API | Python SDK |
|---|---|---|
| Cooldown handling | Manual `time.sleep()` | **Automatic** |
| Error handling | Parse status codes | **Typed exceptions** |
| Type safety | None | **Pydantic models** |
| Multi-character | Manual `asyncio` juggling | **Built-in helpers** |
| Best for | Prototyping, one-off scripts | Bots, long-running programs |

```bash
pip install artifacts-mmo
```

Requires **Python 3.10+**. See [Installation](/sdks/python/introduction/) for full setup instructions.
