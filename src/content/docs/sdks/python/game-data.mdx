title: Game Data
description: Read game data from the Artifacts API using the Python SDK.
---

All game data endpoints are **read-only** (no character required, no cooldown). They return typed Pydantic models and support pagination.

```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    # All examples below use `client`
```

## Pagination

List endpoints return a `DataPage[T]` object:

```python
page = client.items.get_all(page=1, size=50)

page.data    # list[ItemSchema]    — items on this page
page.total   # int                 — total number of matching items
page.page    # int                 — current page number (1-based)
page.pages   # int                 — total number of pages
page.size    # int                 — page size used
```

To iterate over all pages:

```python
page = 1
while True:
    result = client.items.get_all(page=page, size=100)
    for item in result.data:
        print(item.code, item.name)
    if page >= result.pages:
        break
    page += 1
```

## Items

```python
# Get a single item by its code
item = client.items.get("copper_ore")
print(f"{item.name}  lv{item.level}  type={item.type.value}")

# List items with filters
page = client.items.get_all(
    min_level=1,
    max_level=20,
    type="resource",   # "weapon", "armor", "consumable", "resource", "currency", ...
    size=50,
)
for item in page.data:
    print(f"  {item.code}: {item.name}  lv{item.level}")
```

## Monsters

```python
monster = client.monsters.get("chicken")
print(f"{monster.name}  lv{monster.level}  HP={monster.hp}")

# Filter by level range
page = client.monsters.get_all(min_level=1, max_level=10)
for m in page.data:
    print(f"  {m.code}: {m.name}  lv{m.level}")
```

## Maps

```python
# Find all tiles that contain a specific monster
maps = client.maps.get_all(content_type="monster", content_code="chicken")
for tile in maps.data:
    print(f"  ({tile.x},{tile.y})  layer={tile.layer.value}")

# Find all tiles with a bank
banks = client.maps.get_all(content_type="bank")

# Get a specific tile by layer and coordinates
tile = client.maps.get_by_position("overworld", 0, 1)
print(f"Tile ({tile.x},{tile.y}): {tile.content.type.value if tile.content else 'empty'}")

# Get a tile by its database ID
tile = client.maps.get_by_id(42)
```

Common `content_type` values: `"monster"`, `"resource"`, `"workshop"`, `"bank"`, `"grand_exchange"`, `"tasks_master"`, `"npc"`.

## Resources

```python
# List gatherable resources filtered by skill and level
page = client.resources.get_all(skill="mining", min_level=1, max_level=10)
for r in page.data:
    print(f"  {r.code}  lv{r.level}  drops: {[d.code for d in r.drops]}")
```

Available skills: `"woodcutting"`, `"mining"`, `"fishing"`, `"alchemy"`.

## NPCs

```python
# List all NPCs
npcs = client.npcs.get_all()
for npc in npcs.data:
    print(f"  {npc.code}: {npc.name}  type={npc.type.value}")

# Filter NPCs by type, currency, or item
merchants = client.npcs.get_all(type="merchant")

# Get a specific NPC by code
npc = client.npcs.get("merchant_1")
print(f"{npc.name}  type={npc.type.value}")
```

To buy/sell items from NPCs, use `char.trading.npc_buy()` / `char.trading.npc_sell()` — see [Trading](/sdks/python/trading/).

## Events

```python
# Active events right now (exclusive monsters, resources, NPCs)
active = client.events.get_all_active()
for event in active.data:
    print(f"  {event.name}  ends at: {event.expiration}")

# All events (including past ones)
all_events = client.events.get_all()
```

## Achievements

```python
all_achievements = client.achievements.get_all()
for a in all_achievements.data:
    print(f"  {a.code}: {a.name}  (type={a.type.value})")

# Get a single achievement
a = client.achievements.get("first_fight")
```

## Badges, Effects

```python
badges = client.badges.get_all()
effects = client.effects.get_all()
```

## Grand Exchange data

```python
# Currently open orders for an item
orders = client.grand_exchange.get_orders(code="copper_ore")
for order in orders.data:
    print(f"  qty={order.quantity}  price={order.price}  type={order.type.value}")

# Transaction history for an item
history = client.grand_exchange.get_history("copper_ore")
```

## Tasks data

```python
# List available tasks (filterable by type and level)
tasks = client.tasks.get_all(type="monsters", min_level=1)

# List all task rewards
rewards = client.tasks.get_all_rewards()
```

## Leaderboard

```python
# Top characters sorted by combat level (or 'gold', 'woodcutting', etc.)
top_chars = client.leaderboard.get_characters(sort="combat")

# Top accounts
top_accounts = client.leaderboard.get_accounts(sort="gold")
```
