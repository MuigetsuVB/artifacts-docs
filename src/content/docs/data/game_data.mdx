All game data is publicly accessible without authentication. This page is your single reference for every data endpoint, including available filters, response schemas, and examples.

## Pagination

All "Get All" endpoints return paginated results using `DataPage[T]`:

```json title="DataPage[T]"
{
  "data": [ ... ],
  "total": 184,
  "page": 1,
  "size": 50,
  "pages": 4
}
```

| Parameter | Default | Description |
|---|---|---|
| `page` | `1` | Page number (1-indexed). |
| `size` | `50` | Items per page (max 100). |

Use `?page=2&size=100` query parameters to navigate pages.

## Items

Browse all items in the game, filterable by type, subtype, craft skill, name, and more.

**Endpoints:**
- `GET /items` â€” List all items
- `GET /items/{code}` â€” Get a single item by code

**Filters (`GET /items`):**

| Parameter | Type | Description |
|---|---|---|
| `type` | string | Item type: `weapon`, `shield`, `helmet`, `body_armor`, `leg_armor`, `boots`, `ring`, `amulet`, `artifact`, `utility`, `rune`, `bag`, `resource`, `consumable`, `currency`. |
| `craft_skill` | string | Filter by crafting skill: `weaponcrafting`, `gearcrafting`, `jewelrycrafting`, `cooking`, `woodcutting`, `mining`, `alchemy`. |
| `craft_material` | string | Filter by craft material code. |
| `min_level` | int | Minimum item level. |
| `max_level` | int | Maximum item level. |
| `name` | string | Search by item name (partial match). |

<Tabs>
  <TabItem label="cURL">
  ```http
  # Get all weapons
curl -s 'https://api.artifactsmmo.com/items?type=weapon&size=5' \
  -H 'Accept: application/json'

# Get a specific item
curl -s 'https://api.artifactsmmo.com/items/iron_sword' \
  -H 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  // Get all weapons
const res = await fetch('https://api.artifactsmmo.com/items?type=weapon&size=5');
const data = await res.json();
console.log(data);

// Get a specific item
const item = await fetch('https://api.artifactsmmo.com/items/iron_sword');
console.log(await item.json());
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

  # Get all weapons
  resp = requests.get("https://api.artifactsmmo.com/items", params={"type": "weapon", "size": 5})
  print(resp.json())

  # Get a specific item
  resp = requests.get("https://api.artifactsmmo.com/items/iron_sword")
  print(resp.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

  client = ArtifactsClient()
  # Get all weapons
  weapons = client.items.get_all(type="weapon", size=5)
  print(weapons)

  # Get a specific item
  item = client.items.get("iron_sword")
  print(item)
  ```
  </TabItem>
</Tabs>
