Artifacts is a **sandbox MMORPG** where every action is controlled through an **HTTP API**. There is no traditional game client to click — instead, you write code that sends requests to the server, and your characters respond in real time.

## How It Works

1. **Create an account** and up to **5 characters** per account
2. **Get your API token** from [your account page](https://www.artifactsmmo.com/my)
3. **Send HTTP requests** to control your characters — move, fight, gather, craft, trade
4. **Handle cooldowns** — every action has a cooldown you must respect before the next action
5. **Automate everything** — build bots, scripts, or full automation pipelines in any language

The API is fully documented using [OpenAPI](https://api.artifactsmmo.com/docs/). You can also use the official [Python SDK](/sdks/python/) which handles cooldowns, retries, and provides typed models automatically.

## Core Game Loop

There is no fixed path in Artifacts — you choose your own adventure. Typical gameplay loops include:

- **Combat** — Fight [monsters](/concepts/stats_and_fights/) to earn XP, gold, and item drops
- **Gathering** — Harvest [resources](/concepts/skills/) from nodes across the map (mining, woodcutting, fishing, alchemy)
- **Crafting** — Transform resources into [equipment and utilites](/concepts/equipment) at workshops
- **Trading** — Buy and sell items on the [Grand Exchange](/concepts/grand_exchange/) or with [NPCs](/concepts/npcs/)
- **Tasks** — Complete [random objectives](/concepts/tasks/) from task masters for gold and task coins
- **Events** — Participate in [world events](/concepts/events/) for exclusive content

## Seasons

Artifacts runs on a [seasonal server system](/concepts/seasons/). Each season lasts approximately **4 months** and introduces:

- New content like monsters, items and more
- New game mechanics
- Balance changes and quality-of-life improvements
- A full progress reset — everyone starts from scratch

Seasons create a competitive environment where being among the first to unlock achievements earns **unique badges** displayed on your account.

## World Structure

The game world is a 2D grid organized into three **layers**:

| Layer | Description |
|---|---|
| **Overworld** | The main surface — towns, forests, deserts, and oceans |
| **Underground** | Mines, caves, and subterranean areas |
| **Interior** | Buildings, dungeons, and special areas |

You navigate between layers using **transitions** (doors, stairs, portals). See [Maps & Movement](/concepts/maps_and_movement/) for details.

## Character Stats & Progression

Each character has a **combat level** and individual **skill levels** (up to level 50 each). Equipment, runes, and artifacts provide elemental stats for combat. See:

- [Combat & Stats](/concepts/stats_and_fights/) — Combat mechanics, damage formulas, XP
- [Equipment](/concepts/equipment) — Items you can equip to improve your character
- [Skills](/concepts/skills/) — Gathering and crafting skills
- [Resting & Using items](/concepts/resting_and_using_items/) — Recover your health points between fights

## Economy

The economy is divided in two parts:

- [Grand Exchange](/concepts/grand_exchange/) — A marketplace where players post buy and sell orders
- [NPCs](/concepts/npcs/) — Merchants that buy and sell specific items at fixed prices
