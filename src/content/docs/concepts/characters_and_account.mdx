Your account can have up to **5 characters**, each with their own level, inventory, equipment, and position on the map. All characters share a single **bank** and contribute to **account-level achievements**.

When you create an account, you start with no characters. You create them via the API, choosing a unique name and a skin. Characters begin at level 1 at position (0, 0) on the overworld.

## Accounts

Start from the complete account object, then use the endpoints below to create and manage your account.

### Account Model


### Account Endpoints

#### Create an Account

`POST /accounts/create` - Create a new account.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/accounts/create \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --data '{
  "username": "my_account",
  "password": "my_password",
  "email": "player@example.com"
}'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/accounts/create';
  const options = {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      Accept: 'application/json'
    },
    body: JSON.stringify({
      username: 'my_account',
      password: 'my_password',
      email: 'player@example.com'
    })
  };

  try {
    const response = await fetch(url, options);
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error(error);
  }
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

  url = "https://api.artifactsmmo.com/accounts/create"
  headers = {
      "Accept": "application/json",
      "Content-Type": "application/json"
  }
  payload = {
      "username": "my_account",
      "password": "my_password",
      "email": "player@example.com"
  }

  response = requests.post(url, json=payload, headers=headers)
  print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

      created = client.accounts.create(

      )
      print(created)
  ```
  </TabItem>
</Tabs>
