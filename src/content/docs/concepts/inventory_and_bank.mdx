## Inventory

Each character has 20 slots in their inventory, which can hold a maximum of 100 items. So you can have 20 different items, but a maximum of 100 items in total. At each level, the maximum total items increases by 2.

  
```http title="JSON"
....
"inventory": [
  {
    "slot": 0,
    "code": "string",
    "quantity": 0
    }
]
```

## Bank 

At the bank, you can deposit items and gold. The bank is shared with all your characters, it's the best way to trade with your other characters.

The bank is limited to 50 slots, but you can buy expansions at a cost of 4,500 gold for 20 slots, and the price doubles (×2) with each expansion purchased.

### Deposit items


To deposit items, you must be on a map containing a bank.  For more details on map navigation, see [Maps and Movement](/concepts/maps_and_movement/).

:::note
There is a **3-second cooldown** per different item deposit.
:::


To deposit items, use the following request:

**Endpoint:** `POST /my/{name}/action/bank/deposit/item`

<ApiSdkTabs apiUrl="https://api.artifactsmmo.com/docs/#/operations/action_deposit_bank_item_my__name__action_bank_deposit_item_post" sdkUrl="https://python.sdk.artifactsmmo.com/bank.html#deposit-and-withdraw-items">
  <Fragment slot="api-curl">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/bank/deposit/item \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '[
    { "code": "string", "quantity": 1 },
    { "code": "another_item", "quantity": 2 }
]'
  ```
  </Fragment>
  <Fragment slot="api-javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/my/{name}/action/bank/deposit/item';
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify([
    { code: "string", quantity: 1 },
    { code: "another_item", quantity: 2 }
  ])
};

try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </Fragment>
  <Fragment slot="api-python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/{name}/action/bank/deposit/item"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}
data = [
    {"code": "string", "quantity": 1},
    {"code": "another_item", "quantity": 2}
]

response = requests.post(url, json=data, headers=headers)
print(response.json())
  ```
  </Fragment>
  <Fragment slot="sdk-python">
  ```python

    char = client.character("{name}")
    items = [
        SimpleItemSchema(code="string", quantity=1),
        SimpleItemSchema(code="another_item", quantity=2)
    ]
    char.bank.deposit_items(items)
  ```
  </Fragment>
</ApiSdkTabs>

| Field | Description |
|---|---|
| `code` | The item code to deposit. |
| `quantity` | Number of items to deposit. |
