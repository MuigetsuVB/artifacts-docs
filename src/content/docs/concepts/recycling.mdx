Recycling lets you break down crafted equipment to recover some of the original crafting materials. It's useful for clearing unwanted gear or recovering resources.

## Overview

Recycling is available at **weaponcrafting**, **gearcrafting**, and **jewelrycrafting** workshops. Only items that have a craft recipe tied to one of these skills can be recycled.

## How It Works

1. **Move** your character to the appropriate workshop (matching the item's craft skill).
2. **Send** the recycling action with the item code.
3. **Receive** random materials from the item's original craft recipe.

### Requirements

- The item must have a **craft recipe** using `weaponcrafting`, `gearcrafting`, or `jewelrycrafting`.
- Your character must be standing at a **workshop** matching the item's craft skill.
- The item must be in your **inventory** (not equipped).

:::caution
Only `weaponcrafting`, `gearcrafting`, and `jewelrycrafting` items are eligible for recycling. Items crafted with `cooking`, `mining`, `woodcutting`, or `alchemy` cannot be recycled.
:::

### Resources Returned

The materials you receive are chosen **randomly** from the item's craft recipe. The number of resources returned depends on recipe complexity:

```
resources returned = floor((total items in recipe - 1) / 5) + 1
```

With a **minimum of 1 resource** per recycled item.

**Example:** An item with a recipe requiring 8 total materials returns `floor((8 - 1) / 5) + 1 = 2` random resources per item recycled.

### Recycling Details

| Property | Value |
|---|---|
| Skill level required | **None** — any character can recycle |
| Cooldown | 3 seconds per item |
| Multiple at once | Yes, via the `quantity` field |
| Location | Must be at a matching workshop |

## API Endpoints

### Recycle Item

`POST /my/{name}/action/recycling` — Recycle items from your inventory at a workshop.

  
<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/recycling \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
  "code": "iron_sword",
  "quantity": 5
}'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/my/{name}/action/recycling';
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({ code: "iron_sword", quantity: 5 })
};
  
try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/{name}/action/recycling"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}
data = {"code": "iron_sword", "quantity": 5}

response = requests.post(url, json=data, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("{name}")
    char.skills.recycle(code="iron_sword", quantity=5)
  ```
  </TabItem>
</Tabs>

| Field | Description |
|---|---|
| `code` | The item code to recycle. |
| `quantity` | Number of items to recycle (optional, default 1). |
