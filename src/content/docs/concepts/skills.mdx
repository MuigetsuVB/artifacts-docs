Skills allow your character to gather resources and craft items. There are 4 gathering skills and 7 crafting skills, all sharing the same XP-to-level table.

## Gathering

Gathering skills allow you to harvest resources on the map. There are 4 gathering skills: **Woodcutting**, **Mining**, **Fishing**, and **Alchemy**.

### Browse Resources

`GET /resources` — View all available resources and their required skill levels.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request GET \
  --url 'https://api.artifactsmmo.com/resources?page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/resources?page=1&size=50';
const options = {
  method: 'GET',
  headers: { Accept: 'application/json' }
};

try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/resources"
response = requests.get(url, params={"page": 1, "size": 50})
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    page = client.resources.get_all(page=1, size=50)
    for r in page.data:
        print(f"{r.code} (lv{r.level}) drops: {[d.code for d in r.drops]}")
  ```
  </TabItem>
</Tabs>

[View full endpoint details](https://api.artifactsmmo.com/docs/#/operations/get_all_resources_resources_get)

You then need to find a [map](/concepts/maps_and_movement/) that contains this resource.

### Gather Action

`POST /my/{name}/action/gathering` — Gather a resource at your current location. You must have the required skill level.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/gathering \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">

  ```js
  }

try {

  ```

  </TabItem>

  </TabItem>
</Tabs>

:::caution
If your skill level exceeds the resource level by more than **10 levels**, you earn **0 XP** from gathering it.
:::

[View full endpoint details](https://api.artifactsmmo.com/docs/#/operations/action_gathering_my__name__action_gathering_post)


## Crafting

Crafting skills let you make all kinds of items. You can craft with 7 skills: **Weaponcrafting**, **Gearcrafting**, **Jewelrycrafting**, **Cooking**, **Mining**, **Woodcutting**, and **Alchemy**.

Some crafts produce more than one item per craft. Check the `craft.quantity` field on an item to see how many are produced.

### Browse Craftable Items

`GET /items?craft_skill={skill}` — View all items craftable with a specific skill.

### Craft Action

`POST /my/{name}/action/crafting` — Craft an item at a workshop. You must have the required skill level and materials in your inventory.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/crafting \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
  "code": "iron_sword"
}'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/my/{name}/action/crafting';
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  },
  body: JSON.stringify({ code: "iron_sword" })
};

try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/{name}/action/crafting"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}
data = {"code": "iron_sword"}

response = requests.post(url, json=data, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("{name}")
    char.skills.craft(code="iron_sword", quantity=1)
  ```
  </TabItem>
</Tabs>

| Field | Description |
|---|---|
| `code` | The item code to craft. |
| `quantity` | Number of items to craft (optional, default 1). |

:::caution
If your skill level exceeds the craft level by more than **10 levels**, you earn **0 XP** from crafting it.
:::

:::note
Crafting XP varies by skill:
- **Weaponcrafting, Gearcrafting, Jewelrycrafting, Alchemy**: Normal XP (1.0×)
- **Cooking**: Reduced XP (**0.5×**)
- **Mining, Woodcutting, Fishing** (crafting, e.g. bars, planks): Very low XP (**0.1×**) — these skills are primarily leveled through gathering.
:::

[View full endpoint details](https://api.artifactsmmo.com/docs/#/operations/action_crafting_my__name__action_crafting_post)
