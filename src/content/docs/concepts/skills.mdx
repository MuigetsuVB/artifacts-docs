## Gathering  

Gathering skills allow you to gather resources on the map. Currently, 4 gathering skills are available: Woodcutting, Mining, Fishing and Alchemy.

You can see all the resources that can be gathered with this API request.
 
<Tabs>
  <TabItem label="cURL">
  ```http
  curl --location --request GET 'https://api.artifactsmmo.com/resources/?page=1&size=50' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   redirect: 'follow'
};
 
fetch("https://api.artifactsmmo.com/resources/?page=1&size=50", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/resources/?page=1&size=50"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    page = client.resources.get_all(page=1, size=50)
    for r in page.data:
        print(f"{r.code} (lv{r.level}) drops: {[d.code for d in r.drops]}")
  ```
  </TabItem>
</Tabs>

[View API Reference](https://api.artifactsmmo.com/docs/#/operations/get_all_resources_resources__get)

You then need to find a map that contains this resource. [To learn more about the map, click here.](/concepts/maps_and_movement)

When you're on a map containing a resource, if you have the necessary skill level, you can harvest it with this API request.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --location --request POST 'https://api.artifactsmmo.com/my/YOUR_CHARACTER_NAME/action/gathering' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");
myHeaders.append("Authorization", "Bearer YOUR_TOKEN");

   method: 'POST',
   headers: myHeaders,
   redirect: 'follow'
};

   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

url = "https://api.artifactsmmo.com/my/YOUR_CHARACTER_NAME/action/gathering"

    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}

print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("YOUR_CHARACTER_NAME")
    result = char.skills.gather()
    print(f"+{result.details.xp}xp, got: {[d.code for d in result.details.items]}")
  ```
  </TabItem>
</Tabs>

:::caution
If your skill level exceeds the resource level by more than **10 levels**, you earn **0 XP** from gathering it.
:::

[View API Reference](https://api.artifactsmmo.com/docs/#/operations/action_gathering_my__name__action_gathering_post)

 
## Crafting

Crafting skills let you make all kinds of items.  You can craft with 7 skills: Weaponcrafting, Gearcrafting, Jewelrycrafting, Cooking, Mining, Woodcutting and Alchemy.

Some crafts produce more than one item per craft. Check the `craft.quantity` field on an item to see how many are produced.

If you want to see all the crafts of a skill, you can use the Get Item request with the craft_skill parameter. Here's an example request:

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --location --request GET 'https://api.artifactsmmo.com/items/?craft_skill=weaponcrafting' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");

var requestOptions = {
   method: 'GET',
   headers: myHeaders,
   redirect: 'follow'
}; 

fetch("https://api.artifactsmmo.com/items/?craft_skill=weaponcrafting", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/items/?craft_skill=weaponcrafting"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json"
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

with ArtifactsClient(token="YOUR_TOKEN") as client:
    page = client.items.get_all(craft_skill="weaponcrafting")
    for item in page.data:
        print(f"{item.code}: {item.name} (lv{item.level})")
  ```
  </TabItem>
</Tabs>

[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/get_all_items_items__get)


You must then find a map containing a workshop of the skill. [To learn more about the map, click here.](/concepts/maps_and_movement)

When you're on a map containing a workshop, you can use this request to craft an item.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --location -g --request POST 'https://api.artifactsmmo.com/my/{name}/action/crafting' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer YOUR_TOKEN' \
--data-raw '{
  "code": "INSERT_ITEM_CODE"
}'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  var myHeaders = new Headers();
myHeaders.append("Accept", "application/json");
myHeaders.append("Content-Type", "application/json");
myHeaders.append("Authorization", "Bearer YOUR_TOKEN");

var raw = JSON.stringify({
   "code": "INSERT_ITEM_CODE"
});
  
var requestOptions = {
   method: 'POST',
   headers: myHeaders,
   body: raw,
   redirect: 'follow'
};

fetch("https://api.artifactsmmo.com/my/{name}/action/crafting", requestOptions)
   .then(response => response.text())
   .then(result => console.log(result))
   .catch(error => console.log('error', error));
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/{name}/action/crafting"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}
data = {"code": "INSERT_ITEM_CODE"}

response = requests.post(url, json=data, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("{name}")
    char.skills.craft(code="INSERT_ITEM_CODE", quantity=1)
  ```
  </TabItem>
</Tabs>

| Field | Description |
|---|---|
| `code` | The item code to craft. |
| `quantity` | Number of items to craft (optional, default 1). |

:::caution
If your skill level exceeds the craft level by more than **10 levels**, you earn **0 XP** from crafting it.
:::

:::note
Crafting XP varies by skill:
- **Weaponcrafting, Gearcrafting, Jewelrycrafting, Alchemy**: Normal XP (1.0×)
- **Cooking**: Reduced XP (**0.5×**)
- **Mining, Woodcutting, Fishing** (crafting, e.g. bars, planks): Very low XP (**0.1×**) — these skills are primarily leveled through gathering.
:::

[View the API Reference](https://api.artifactsmmo.com/docs/#/operations/action_crafting_my__name__action_crafting_post)
