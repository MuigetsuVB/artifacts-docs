NPCs are merchant or trader characters from whom you can buy, sell or trade items. Some are static, while others appear dynamically thanks to the [event system](/concepts/events). 

## NPC Types

| Type | Description |
|------|-------------|
| `merchant` | Buys and sells items for gold. |
| `trader` | Trades items for other items (currency is an item code instead of gold). |

When viewing the items on an NPC, if the `currency` field shows anything other than `gold`, it will display the code of the item you need to trade to the NPC.

## List of NPCs

To view the list of all existing NPCs, use the following request:

**Endpoint:** `GET /npcs/details`

<ApiSdkTabs apiUrl="https://api.artifactsmmo.com/docs/#/operations/get_all_npcs_npcs_details_get" sdkUrl="https://python.sdk.artifactsmmo.com/game-data.html#npcs">
  <Fragment slot="api-curl">
  ```http
  curl --location --request GET 'https://api.artifactsmmo.com/npcs/details' \
--header 'Accept: application/json' \
--header 'Content-Type: application/json'
  ```
  </Fragment>
  <Fragment slot="api-javascript">
  ```js

const options = {

  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);

  console.error(error);

  ```
  </Fragment>
  <Fragment slot="api-python">
  ```python

    "Accept": "application/json",
    "Content-Type": "application/json"
}

print(response.json())
  ```
  </Fragment>
  <Fragment slot="sdk-python">
  ```python

with ArtifactsClient(token="YOUR_TOKEN") as client:
    npcs = client.npcs.get_all()
    for npc in npcs.data:
        print(f"{npc.code} ({npc.type.value})")
  ```
  </Fragment>
</ApiSdkTabs>

| Parameter | Description |
|---|---|
| `name` | NPC name. |
| `type` | Type of NPCs. |
| `currency` | Currency code to filter NPCs that trade with this currency. |
| `item` | Item code to filter NPCs that trade this item. |
| `page` | Page number (default: 1). |
| `size` | Page size (default: 50). |

## Actions

To buy or sell an item, you must be on a map with an NPC. For more information, see [Maps & Movement](/concepts/maps_and_movement/).

### Buy Item

To buy an item from an NPC, use the following request:

**Endpoint:** `POST /my/{name}/action/npc/buy`

<ApiSdkTabs apiUrl="https://api.artifactsmmo.com/docs/#/operations/action_npc_buy_item_my__name__action_npc_buy_post" sdkUrl="https://python.sdk.artifactsmmo.com/trading.html#buy-from-an-npc">
  <Fragment slot="api-curl">
  ```http
  curl --request POST \
  --url https://api.artifactsmmo.com/my/{name}/action/npc/buy \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '{
  "code": "string",
  "quantity": 1
}'
  ```
  </Fragment>
  <Fragment slot="api-javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/my/{name}/action/npc/buy';
const options = {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  },
    body: '{"code":"string","quantity":1}'

};
  
try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </Fragment>
  <Fragment slot="api-python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/{name}/action/npc/buy"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}
data = {"code": "string", "quantity": 1}

response = requests.post(url, json=data, headers=headers)
print(response.json())
  ```
  </Fragment>
  <Fragment slot="sdk-python">
  ```python
  from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("{name}")
    char.trading.npc_buy(code="wooden_staff", quantity=1)
  ```
  </Fragment>
</ApiSdkTabs>

| Field | Description |
|---|---|
| `code` | The item code to buy. |
| `quantity` | Number of items to buy (1–100). |
