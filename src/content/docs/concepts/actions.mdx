Every interaction your character makes with the game world is an **action** â€” moving, fighting, crafting, trading, and more. Each action is a `POST` request to `/my/{name}/action/...` and returns the result immediately along with a cooldown.

## Overview

### How Actions Work

1. Send a `POST` request to the appropriate action endpoint.
2. The API returns the action result and a `CooldownSchema` in the response.
3. Your character enters a cooldown period â€” you must wait before sending another action for that character.
4. If you send an action while in cooldown, you receive error **499** with the remaining time.

### CooldownSchema

Every action response includes a `cooldown` object:

```json title="CooldownSchema"
{
  "cooldown": {
    "total_seconds": 5,
    "remaining_seconds": 5,
    "started_at": "2025-01-15T12:00:00Z",
    "expiration": "2025-01-15T12:00:05Z",
    "reason": "movement"
  }
}
```

| Field | Description |
|---|---|
| `total_seconds` | Total cooldown duration in seconds. |
| `remaining_seconds` | Seconds remaining until the cooldown expires. |
| `started_at` | When the cooldown started (ISO 8601). |
| `expiration` | When the cooldown expires (ISO 8601). |
| `reason` | What caused the cooldown (e.g., `movement`, `fight`, `crafting`). |

:::tip
You can use the `expiration` field to schedule your next action precisely. Simply wait until the current time reaches the expiration timestamp.
:::

## API Endpoints

### Action Logs

`GET /my/logs/{name}` â€” View all logged actions for a character.

Every action your character performs is recorded. Logs are paginated and include the action type, description, timestamp, and result.

<Tabs>
  <TabItem label="cURL">
  ```http
  curl --request GET \
  --url 'https://api.artifactsmmo.com/my/logs/{name}?page=1&size=50' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
  const url = 'https://api.artifactsmmo.com/my/logs/{name}?page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  }
};

try {
  const response = await fetch(url, options);
  const data = await response.json();
  console.log(data);
} catch (error) {
  console.error(error);
}
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

url = "https://api.artifactsmmo.com/my/logs/{name}"
headers = {
    "Accept": "application/json",
    "Authorization": "Bearer YOUR_TOKEN"
}

response = requests.get(url, params={"page": 1, "size": 50}, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

with ArtifactsClient(token="YOUR_TOKEN") as client:
    char = client.character("{name}")
    logs = char.get_logs(page=1, size=50)
    for log in logs.data:
        print(f"[{log.type.value}] {log.description}")
  ```
  </TabItem>
