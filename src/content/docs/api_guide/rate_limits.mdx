# Rate limits

## Rate Limit Values

| Type       | /second    | /minute    | /hour       |
|------------|------------|------------|-------------|
| `account`  | 10/s       | —          | 500/h       |
| `data`     | 20/s       | 500/min    | 10 000/h    |
| `action`   | 20/2s      | 500/min    | 10 000/h    |
