The Artifacts API is a RESTful HTTP API that accepts and returns JSON. Every game action — from moving a character to crafting an item — maps to an API endpoint.

## Base URL

```
https://api.artifactsmmo.com
```

All endpoints are relative to this base. For example, the move action is:

```
POST https://api.artifactsmmo.com/my/{name}/action/move
```

:::tip
Other servers are available:
- **Sandbox** (members): `https://api.sandbox.artifactsmmo.com`
- **Test** (members): `https://api.test.artifactsmmo.com`
:::

## Request Format

- **Content-Type**: `application/json` for all `POST` request bodies
- **Accept**: `application/json` for all responses
- **Authentication**: `Authorization: Bearer YOUR_TOKEN` (required for authenticated endpoints)

### Example Request

<Tabs>
  <TabItem label="cURL">
  ```bash
  curl -X POST https://api.artifactsmmo.com/my/MyChar/action/move \
    -H "Content-Type: application/json" \
    -H "Accept: application/json" \
    -H "Authorization: Bearer YOUR_TOKEN" \
    -d '{"x": 0, "y": 1}'
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
  import requests

  response = requests.post(
      "https://api.artifactsmmo.com/my/MyChar/action/move",
      json={"x": 0, "y": 1},
      headers={
          "Content-Type": "application/json",
          "Authorization": "Bearer YOUR_TOKEN",
      },
  )
  print(response.json())
  ```
  </TabItem>
</Tabs>

## Response Format

All responses return JSON with a `data` wrapper:

```json title="Success Response"
{
  "data": {
    // Response payload
  }
}
```

Action endpoints also include a `cooldown` object:

```json title="Action Response"
{
  "data": {
    "cooldown": {
      "total_seconds": 5,
      "remaining_seconds": 5,
      "started_at": "2025-01-15T12:00:00Z",
      "expiration": "2025-01-15T12:00:05Z",
      "reason": "movement"
    },
    "character": { /* updated character state */ }
  }
}
```

## Pagination

All list endpoints support pagination with these query parameters:

| Parameter | Type | Default | Description |
|---|---|---|---|
| `page` | int | 1 | Page number (1-indexed) |
| `size` | int | 50 | Items per page (max 100) |

Paginated responses include metadata:

```json
{
  "data": [ /* array of items */ ],
  "total": 150,
  "page": 1,
  "size": 50,
  "pages": 3
}
```

## Endpoint Categories

The API is organized into these categories:

### Public Endpoints (no auth required)

| Category | Base Path | Description |
|---|---|---|
| Server | `/` | Server status and version |
| Items | `/items` | Browse all game items |
| Monsters | `/monsters` | Browse all monsters |
| Maps | `/maps` | Browse the world map |
| Resources | `/resources` | Browse gatherable resources |
| NPCs | `/npcs` | Browse NPCs and their inventories |
| Events | `/events` | View active and past events |
| Achievements | `/achievements` | Browse all achievements |
| Badges | `/badges` | Browse all badges |
| Effects | `/effects` | Browse status effects |
| Tasks | `/tasks` | Browse task types and rewards |
| Grand Exchange | `/grandexchange` | View marketplace orders and history |
| Leaderboard | `/leaderboard` | Character and account rankings |
| Accounts | `/accounts` | View public account info |
| Characters | `/characters` | View public character info |

### Authenticated Endpoints (token required)

| Category | Base Path | Description |
|---|---|---|
| Token | `/token` | Generate authentication tokens |
| My Account | `/my` | View your account, bank, and characters |
| Character Actions | `/my/{name}/action/*` | Perform all character actions |
| Character Logs | `/my/logs` | View action history |

### Members-Only Endpoints

| Category | Base Path | Description |
|---|---|---|
| Simulation | `/simulation` | Fight simulator |
| Sandbox | `/sandbox` | Sandbox server tools |
