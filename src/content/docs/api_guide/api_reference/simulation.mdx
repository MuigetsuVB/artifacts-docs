### Fight Simulation

**Endpoint:** `POST /simulation/fight_simulation`

**Authentication:** Bearer token required

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| body.characters | body | array&lt;FakeCharacterSchema&gt; | Yes | List of fake characters (1-3). |
| body.monster | body | string | Yes | Monster code to fight against. |
| body.iterations | body | integer | Yes | Number of combat iterations to simulate. |
