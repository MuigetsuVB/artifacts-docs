### Get All Tasks

**Endpoint:** `GET /tasks/list`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| min_level | query | integer | No | Minimum level. |
| max_level | query | integer | No | Maximum level. |
| skill | query | enum(weaponcrafting, gearcrafting, jewelrycrafting, cooking, ...) | No | Skill of tasks. |
| type | query | enum(monsters, items) | No | Type of tasks. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/tasks/list?min_level=1&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/tasks/list?min_level=1&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

url = "https://api.artifactsmmo.com/tasks/list?min_level=1&page=1&size=50"

    "Accept": "application/json",
}

print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">

  </TabItem>

### Get Task

**Endpoint:** `GET /tasks/list/{code}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
