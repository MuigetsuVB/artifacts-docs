### Get All Maps

**Endpoint:** `GET /maps`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| layer | query | enum(interior, overworld, underground) | No | Filter maps by layer. |
| content_type | query | enum(monster, resource, workshop, bank, ...) | No | Type of maps. |
| content_code | query | string | No | Content code on the map. |
| hide_blocked_maps | query | boolean | No | When true, excludes maps with access_type 'blocked' from the results. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/maps?layer=interior&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/maps?layer=interior&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

url = "https://api.artifactsmmo.com/maps?layer=interior&page=1&size=50"
headers = {
    "Accept": "application/json",
}

print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

    result = client.maps.get_all(page=1, size=50)
    print(result)
  ```
  </TabItem>
</Tabs>

**Authentication:** Public endpoint

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| layer | path | enum(interior, overworld, underground) | Yes | The layer of the map (interior, overworld, underground). |
| content_type | query | enum(monster, resource, workshop, bank, ...) | No | Type of maps. |

### Get Map By Position

**Endpoint:** `GET /maps/{layer}/{x}/{y}`

**Authentication:** Public endpoint
