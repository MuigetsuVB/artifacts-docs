### Get All Events

**Endpoint:** `GET /events`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| type | query | enum(monster, resource, workshop, bank, ...) | No | Type of events. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/events?type=monster&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/events?type=monster&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/events?type=monster&page=1&size=50"
headers = {
    "Accept": "application/json",
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

with ArtifactsClient() as client:
    result = client.events.get_all(page=1, size=50)
    print(result)
  ```
  </TabItem>
</Tabs>

### Spawn Event

**Endpoint:** `POST /events/spawn`

**Authentication:** Bearer token required

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| body.code | body | string | Yes | Code of the event to spawn |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request POST \
  --url 'https://api.artifactsmmo.com/events/spawn' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '  {
    "code": "copper_ore"
  }
  '
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/events/spawn';
const options = {
  method: 'POST',
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    code: "copper_ore"
  })
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/events/spawn"
headers = {
    "Accept": "application/json",
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json",
}

payload = {
    "code": "copper_ore"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
  ```
  </TabItem>
