## Event Tokens

Members can use **event tokens** to spawn world events on demand. Event tokens are distributed periodically and can trigger special events that are normally random.

:::note
Event tokens are a limited resource. Each token spawns one event instance that lasts for a set duration and is visible to all players.
:::

## Fight Simulator

The fight simulator lets you **test combat scenarios** without risking your character. Create fake characters with custom stats and equipment, then simulate battles against any monster in the game.

```python
from artifacts.models.sandbox import FakeCharacterSchema

fake = FakeCharacterSchema(
    level=20,
    weapon_slot="iron_sword",
    body_armor_slot="iron_armor",
)

result = client.simulation.fight(
    characters=[fake],
    monster="ogre",
    iterations=100,
)
print(f"Winrate: {result.winrate:.1%} ({result.wins}W / {result.losses}L)")
```

**Key features:**
- Test with up to 3 characters at once (boss fight simulation)
- Run hundreds of iterations for statistical accuracy
- Try different equipment loadouts without crafting
- Available via `POST /simulation/fight` — [view endpoint](https://api.artifactsmmo.com/docs/#/operations/simulate_fight_simulation_fight_post)

## Sandbox Server

A separate server where members can freely experiment. The sandbox provides **cheat commands** for rapid testing:

| Command | Description |
|---|---|
| Give gold | Add gold to a character |
| Give item | Add any item to a character |
| Give XP | Add XP to any skill on a character |
| Spawn event | Trigger a world event |
| Reset account | Wipe all progress and start fresh |

**Sandbox URL:** `https://api.sandbox.artifactsmmo.com`

```python
from artifacts import ArtifactsClient

client = ArtifactsClient(
    token="your_token",
    base_url="https://api.sandbox.artifactsmmo.com",
)

client.sandbox.give_gold("MyChar", 50000)
client.sandbox.give_item("MyChar", "fire_sword", 1)
client.sandbox.give_xp("MyChar", "mining", 10000)
```

:::caution
The sandbox server is completely separate from the production server. Progress, characters, and items do not transfer between servers.
:::

## Test Server

The test server is used for previewing **upcoming features** before they go live. It runs a development version of the API and may be unstable.

**Test URL:** `https://api.test.artifactsmmo.com`

Content on the test server may change without notice and is typically reset before each season.
