:::

The Artifacts Realtime service delivers **push notifications** over WebSockets, so your bot can react to game events instantly instead of polling the API.

## What You Can Receive

| Notification | Code | Category | Description |
|---|---|---|---|
| Event spawn | `event_spawn` | Events | A world event has spawned |
| Event removed | `event_removed` | Events | A world event has expired |
| GE sell order | `grandexchange_sell_order` | Grand Exchange | A new sell order was created |
| GE buy order | `grandexchange_buy_order` | Grand Exchange | A new buy order was created |
| GE buy completed | `grandexchange_buy` | Grand Exchange | A purchase was completed |
| GE sell completed | `grandexchange_sell` | Grand Exchange | A sale was completed |
| Pending item | `pending_item_received` | Pending Items | An item was sent to your pending queue |
| Online characters | `online_characters` | Characters | List of online characters (every 10 min) |
| Version update | `version` | Server | API server version on connect and updates |
| Announcement | `announcement` | Server | A new server announcement |
| Achievement unlock | `achievement_unlocked` | Account | An achievement was unlocked on your account |
| Account log | `account_log` | Account | An action was performed by one of your characters |
| Test | `test` | Test | Test message every 60 seconds |

## Server URL

```
wss://realtime.artifactsmmo.com
```

## Quick Start

1. Connect to `wss://realtime.artifactsmmo.com`
2. Send your authentication message with your token and desired subscriptions
3. Receive push notifications as JSON messages

See [Connection & Auth](/realtime/connection/) for full examples in Python, JavaScript, and Java.

## Use Cases

- **Market bot** — React instantly to new Grand Exchange orders
- **Event hunter** — Get notified when rare events spawn
- **Dashboard** — Live-update a monitoring dashboard with character activity
- **Multi-character coordination** — Use account logs to coordinate between characters
