:::caution
Sandbox Server is a feature reserved for members. You can sign up for a subscription in your [member area](https://www.artifactsmmo.com/account/).
:::

The sandbox server provides a dedicated environment for testing and developing scripts alongside the official server. It offers additional endpoints to validate any scenario without impacting your main account.

To access the Sandbox server from the client, select the server on the login screen.
<Image src={sandbox4} alt="sandbox4" />


Sandbox features are also available in the client:
<Image src={sandbox1} alt="sandbox1" />
<Image src={sandbox3} alt="sandbox3" />



| Server         | URL |
|--------------|-------------------|
| API server      | https://api.sandbox.artifactsmmo.com/     |
| Websockets server | wss://realtime.sandbox.artifactsmmo.com     |
| Client (Change the server to Sandbox in the bottom-left corner of the login screen) | https://play.artifactsmmo.com   |
