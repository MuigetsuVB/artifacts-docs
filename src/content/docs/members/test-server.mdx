title: Beta Server
---

:::caution
Beta Server is a feature reserved for members. You can sign up for a subscription in your [member area](https://www.artifactsmmo.com/account/).
:::

The beta server is a temporary server that allows you to test the new Artifacts season before its official release, usually about a week before the season launches.

The encyclopedia, public API, and documentation **are available for free to everyone**. However, to connect to the server and use the sandbox features, you must be a member.
 
| Server         | URL |
|--------------|-------------------|
| API server      | https://api.beta.artifactsmmo.com/     |
| API Documentation  | https://api.beta.artifactsmmo.com/docs/#/      |
| Websockets server | wss://realtime.beta.artifactsmmo.com     |
| Documentation  | https://docs.beta.artifactsmmo.com/     |
| Site/Encyclopedia  | https://beta.artifactsmmo.com/       |


It is highly recommended to join [the Discord server](https://discord.gg/prEBQ8a6Vs) to connect with the community and report bugs.


## Sandbox features
Sandbox features are enabled on the beta server, allowing you to:
 * [Give Gold](https://api.sandbox.artifactsmmo.com/docs/#/operations/give_gold_sandbox_give_gold_post)
 * [Give Item](https://api.sandbox.artifactsmmo.com/docs/#/operations/give_item_sandbox_give_item_post)
 * [Give XP](https://api.sandbox.artifactsmmo.com/docs/#/operations/give_xp_sandbox_give_xp_post)
 * [Spawn Event](https://api.sandbox.artifactsmmo.com/docs/#/operations/spawn_event_sandbox_spawn_event_post)
 * [Clear Cooldown](https://api.sandbox.artifactsmmo.com/docs/#/operations/clear_cooldown_sandbox_clear_cooldown_post)
 * [Teleport](https://api.sandbox.artifactsmmo.com/docs/#/operations/teleport_sandbox_teleport_post)
 * [Reset Account](https://api.sandbox.artifactsmmo.com/docs/#/operations/reset_account_sandbox_reset_account_post)
