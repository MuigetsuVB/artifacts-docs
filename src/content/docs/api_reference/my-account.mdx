### Get Bank Details

**Endpoint:** `GET /my/bank`

**Authentication:** Bearer token required

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| None | - | - | - | This endpoint has no parameters. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/my/bank' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js

  method: 'GET',
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

    "Accept": "application/json",

  ```
  </TabItem>

### Change Password

**Endpoint:** `POST /my/change_password`

**Authentication:** Bearer token required

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| body.current_password | body | string | Yes | Your password. |
| body.new_password | body | string | Yes | New password. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request POST \
  --url 'https://api.artifactsmmo.com/my/change_password' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '  {
    "current_password": "YOUR_PASSWORD",
    "new_password": "YOUR_PASSWORD"
  }
  '
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/my/change_password';
const options = {
  method: 'POST',
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    current_password: "YOUR_PASSWORD",
    new_password: "YOUR_PASSWORD"
  })
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/my/change_password"
headers = {
    "Accept": "application/json",
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json",
}

payload = {
    "current_password": "YOUR_PASSWORD",
    "new_password": "YOUR_PASSWORD"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
from artifacts import ArtifactsClient

with ArtifactsClient(token="YOUR_TOKEN") as client:
    result = client.my_account.change_password(password="OLD_PASSWORD", new_password="NEW_PASSWORD")
    print(result)
  ```
  </TabItem>
</Tabs>

[View API Reference](https://api.artifactsmmo.com/docs/#/operations/change_password_my_change_password_post)
