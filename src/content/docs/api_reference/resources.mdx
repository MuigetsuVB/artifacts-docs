### Get All Resources

**Endpoint:** `GET /resources`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| min_level | query | integer | No | Minimum level. |
| max_level | query | integer | No | Maximum level. |
| skill | query | enum(mining, woodcutting, fishing, alchemy) | No | Skill of resources. |
| drop | query | string | No | Item code of the drop. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/resources?min_level=1&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/resources?min_level=1&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">

  </TabItem>

### Get Resource

**Endpoint:** `GET /resources/{code}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| code | path | string | Yes | The code of the resource. |

#### Examples
