### Get Character

**Endpoint:** `GET /characters/{name}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| name | path | string | Yes | The name of the character. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/characters/YOUR_CHARACTER_NAME' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/characters/YOUR_CHARACTER_NAME';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

    "Accept": "application/json",

  ```
  </TabItem>

### Create Character

**Endpoint:** `POST /characters/create`

**Authentication:** Bearer token required

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| body.name | body | string | Yes | Your desired character name. It's unique and all players can see it. |
| body.skin | body | enum(men1, men2, men3, women1, ...) | Yes | Your desired skin. Skins unlocked by default: 'men1', 'men2', 'men3', 'women1', 'women2', 'women3'. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request POST \
  --url 'https://api.artifactsmmo.com/characters/create' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer YOUR_TOKEN' \
  --header 'Content-Type: application/json' \
  --data '  {
    "name": "YOUR_CHARACTER_NAME",
    "skin": "men1"
  }
  '
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/characters/create';
const options = {
  method: 'POST',
  headers: {
    Accept: 'application/json',
    Authorization: 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: "YOUR_CHARACTER_NAME",
    skin: "men1"
  })
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/characters/create"
headers = {
    "Accept": "application/json",
    "Authorization": "Bearer YOUR_TOKEN",
    "Content-Type": "application/json",
}

payload = {
    "name": "YOUR_CHARACTER_NAME",
    "skin": "men1"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
from artifacts import ArtifactsClient
from artifacts.models import CharacterSkin

with ArtifactsClient(token="YOUR_TOKEN") as client:
    result = client.characters.create(name="YOUR_CHARACTER_NAME", skin=CharacterSkin.MEN1)
    print(result)
  ```
  </TabItem>
</Tabs>
