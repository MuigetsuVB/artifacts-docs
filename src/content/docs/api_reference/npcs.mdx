### Get All Npcs

**Endpoint:** `GET /npcs/details`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| name | query | string | No | NPC name. |
| type | query | enum(merchant, trader) | No | Type of NPCs. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/npcs/details?name=YOUR_CHARACTER_NAME&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/npcs/details?name=YOUR_CHARACTER_NAME&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/npcs/details?name=YOUR_CHARACTER_NAME&page=1&size=50"
headers = {
    "Accept": "application/json",
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

    result = client.npcs.get_all(page=1, size=50)
    print(result)
  ```
  </TabItem>
</Tabs>

### Get All Npcs Items

**Endpoint:** `GET /npcs/items`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
