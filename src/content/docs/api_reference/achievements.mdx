### Get All Achievements

**Endpoint:** `GET /achievements`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| type | query | enum(combat_kill, combat_drop, combat_level, gathering, ...) | No | Type of achievements. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/achievements?type=combat_kill&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/achievements?type=combat_kill&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/achievements?type=combat_kill&page=1&size=50"
headers = {
    "Accept": "application/json",
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
from artifacts import ArtifactsClient

with ArtifactsClient() as client:
    result = client.achievements.get_all(page=1, size=50)
    print(result)
  ```
  </TabItem>
</Tabs>

### Get Achievement

**Endpoint:** `GET /achievements/{code}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| code | path | string | Yes | The code of the achievement. |

#### Examples

<Tabs>
