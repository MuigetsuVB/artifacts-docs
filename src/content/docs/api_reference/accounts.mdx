### Get Account

**Endpoint:** `GET /accounts/{account}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| account | path | string | Yes | The name of the account. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/accounts/YOUR_ACCOUNT' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/accounts/YOUR_ACCOUNT';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/accounts/YOUR_ACCOUNT"
headers = {
    "Accept": "application/json",
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python

    result = client.accounts.get("YOUR_ACCOUNT")
  ```

  </TabItem>
</Tabs>

### Get Account Achievements

**Endpoint:** `GET /accounts/{account}/achievements`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| account | path | string | Yes | The name of the account. |
| type | query | enum(combat_kill, combat_drop, combat_level, gathering, ...) | No | Type of achievements. |
| completed | query | boolean | No | Filter by completed achievements. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  ```

### Create Account

**Endpoint:** `POST /accounts/create`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| body.username | body | string | Yes | Your desired username. |
| body.password | body | string | Yes | Your password. |
| body.email | body | string (email) | Yes | Your email. |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request POST \
  --url 'https://api.artifactsmmo.com/accounts/create' \
  --header 'Accept: application/json' \
  --header 'Content-Type: application/json' \
  --data '  {
    "username": "YOUR_USERNAME",
    "password": "YOUR_PASSWORD",
    "email": "you@example.com"
  }
  '
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/accounts/create';
const options = {
  method: 'POST',
  headers: {
    Accept: 'application/json',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    username: "YOUR_USERNAME",
    password: "YOUR_PASSWORD",
    email: "you@example.com"
  })
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/accounts/create"
headers = {
    "Accept": "application/json",
    "Content-Type": "application/json",
}

payload = {
    "username": "YOUR_USERNAME",
    "password": "YOUR_PASSWORD",
    "email": "you@example.com"
}

response = requests.post(url, headers=headers, json=payload)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
from artifacts import ArtifactsClient

with ArtifactsClient() as client:
    result = client.accounts.create(username="YOUR_USERNAME", password="YOUR_PASSWORD", email="you@example.com")
    print(result)
  ```
  </TabItem>
</Tabs>

[View API Reference](https://api.artifactsmmo.com/docs/#/operations/create_account_accounts_create_post)
