### Get All Items

**Endpoint:** `GET /items`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| name | query | string | No | Name of the item. |
| min_level | query | integer | No | Minimum level. |
| max_level | query | integer | No | Maximum level. |
| type | query | enum(utility, body_armor, weapon, resource, ...) | No | Type of items. |
| craft_skill | query | enum(weaponcrafting, gearcrafting, jewelrycrafting, cooking, ...) | No | Skill to craft items. |
| craft_material | query | string | No | Item code of items used as material for crafting. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

### Get Item

**Endpoint:** `GET /items/{code}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| code | path | string | Yes | The code of the item. |
