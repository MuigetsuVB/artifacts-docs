### Get Ge History

**Endpoint:** `GET /grandexchange/history/{code}`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| code | path | string | Yes | The code of the item. |
| account | query | string | No | Account involved in the transaction (matches either seller or buyer). |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/grandexchange/history/copper_ore?account=YOUR_ACCOUNT&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/grandexchange/history/copper_ore?account=YOUR_ACCOUNT&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python

  ```
  </TabItem>

### Get Ge Orders

**Endpoint:** `GET /grandexchange/orders`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| code | query | string | No | The code of the item. |
| account | query | string | No | The account that sells or buys items. |
| type | query | enum(sell, buy) | No | Filter by order type (sell or buy). |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |
