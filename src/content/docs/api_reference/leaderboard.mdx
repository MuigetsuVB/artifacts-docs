### Get Accounts Leaderboard

**Endpoint:** `GET /leaderboard/accounts`

**Authentication:** Public endpoint

#### Parameters

| Parameter | In | Type | Required | Description |
|---|---|---|---|---|
| sort | query | enum(achievements_points, gold) | No | Sort of account leaderboards. |
| name | query | string | No | Account name. |
| page | query | integer | No | Page number |
| size | query | integer | No | Page size |

#### Examples

<Tabs>
  <TabItem label="cURL">
  ```bash
curl --request GET \
  --url 'https://api.artifactsmmo.com/leaderboard/accounts?sort=achievements_points&page=1&size=50' \
  --header 'Accept: application/json'
  ```
  </TabItem>
  <TabItem label="JavaScript" icon="seti:javascript">
  ```js
const url = 'https://api.artifactsmmo.com/leaderboard/accounts?sort=achievements_points&page=1&size=50';
const options = {
  method: 'GET',
  headers: {
    Accept: 'application/json'
  }
};

const response = await fetch(url, options);
const data = await response.json();
console.log(data);
  ```
  </TabItem>
  <TabItem label="Python" icon="seti:python">
  ```python
import requests

url = "https://api.artifactsmmo.com/leaderboard/accounts?sort=achievements_points&page=1&size=50"
headers = {
    "Accept": "application/json",
}

response = requests.get(url, headers=headers)
print(response.json())
  ```
  </TabItem>
  <TabItem label="Python SDK" icon="seti:python">
  ```python
from artifacts import ArtifactsClient

with ArtifactsClient() as client:
    result = client.leaderboard.get_accounts(sort="gold", page=1, size=50)
    print(result)
  ```
  </TabItem>
</Tabs>
